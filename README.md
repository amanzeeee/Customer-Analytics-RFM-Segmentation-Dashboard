##  Project Overview

This project focuses on customer behavior analysis using **RFM (Recency, Frequency, Monetary) Segmentation**. The objective is to identify valuable customers, understand purchasing patterns, and generate actionable business insights that support customer retention and targeted marketing strategies.

The solution was built using **MySQL, Power BI, DAX, and Excel**, covering the complete analytics workflow from data preparation and customer segmentation to dashboard development.

---

##  Business Problem

Businesses often struggle to identify their most valuable customers and recognize customers who are at risk of churning.

This project applies the **RFM (Recency, Frequency, Monetary)** model to segment customers into meaningful groups, enabling businesses to:

- Identify high-value customers
- Improve customer retention
- Design targeted marketing campaigns
- Increase customer lifetime value
- Support data-driven decision making

##  Project Workflow

### 1. Data Preparation
- Cleaned transaction data
- Removed invalid records
- Created sales calculations
- Standardized data types

### 2. SQL Analysis
- Calculated Recency
- Calculated Frequency
- Calculated Monetary Value
- Generated RFM Scores using SQL
- Classified customers into business segments

### 3. Customer Segmentation

Customers were segmented into:

- Champions
- Loyal Customers
- Potential Loyalists
- At Risk
- Others

### 4. Dashboard Development

Designed an interactive Power BI dashboard featuring:

- Customer KPIs
- Monthly Active Customers
- Customer Segment Distribution
- Customer Spending Distribution
- Customer Recency Distribution
- Average Customer Value by Segment
- Purchase Frequency by Segment
- Top 10 Customers by Revenue
